# Personal Portfolio Website PRD

## Project Overview

### Project Name
Akash R — Personal Website

### Project Type
Personal Branding Portfolio Website

### Primary Goal
Build a high-quality personal website that establishes Akash as a Product Designer while showcasing selected work, skills, thinking process, and personality.

### Success Criteria
- Pixel-perfect implementation of Figma design
- Responsive experience across Mobile, Tablet, and Desktop
- Smooth modern interactions
- Lighthouse score above 90
- SEO optimized
- Fast loading performance
- Deployed on Vercel

## Product Vision

The website should feel like a thoughtfully crafted digital representation of Akash rather than a traditional portfolio.

## Technology Stack
- Next.js 15
- TypeScript
- Tailwind CSS
- Framer Motion
- Vercel

## Design Source of Truth
